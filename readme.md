### react 路由的作用
用react-router-dom可以创建单页应用。可以将组件映射到路由上，将对应的组件渲染到想要渲染的位置。根据路径的变化
重新渲染组件。

### react路由的两种实现方式：
- HashRouter：利用hash实现路由切换
- BrowserRouter：利用h5 Api实现路由的切换

### 需要考虑的问题
- 使用context跨组件传递 `history`，`location`，`match`属性传递给包裹的组件。
